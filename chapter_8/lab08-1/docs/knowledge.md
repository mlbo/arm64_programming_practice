# 知识点：ARM64 ABI 调用约定

## 1. 参数传递

| 寄存器 | 用途 |
|--------|------|
| x0-x7 | 参数 1-8 |
| x0 | 返回值 |

## 2. 被调用者保存寄存器

必须由被调用函数保存的寄存器：
- x19-x28
- x29 (FP)
- x30 (LR)

## 3. 调用者保存寄存器

可以由调用者保存的寄存器：
- x0-x18
- x30 (如果是非叶子函数)

## 4. 栈对齐

- 栈必须 16 字节对齐
- 函数调用时 SP 必须满足对齐要求

## 5. 汇编调用 C

```asm
// C 函数: int add(int a, int b);
mov x0, #10          // 参数 1
mov x1, #20          // 参数 2
bl add               // 调用
// x0 = 30           // 返回值
```

## 6. C 调用汇编

```c
// 声明汇编函数
extern int compare(int a, int b);

// 调用
int result = compare(5, 3);
```
