# Lab 05-3-Fixed: BL 指令与返回地址 (正确实现)

## 实验目的

1. 理解如何正确保护返回地址
2. 学习两种保存 x30 的方法

## 修复方案

### 方案一：使用临时寄存器

```asm
bl_test:
    mov x6, x30           // 保存返回地址到 x6
    mov x0, 1
    mov x1, 3
    bl csel_test
    mov x30, x6           // 恢复返回地址
    ret
```

**优点**: 简单，无需栈操作
**缺点**: 占用寄存器，仅适用于简单场景

### 方案二：使用栈 (推荐)

```asm
bl_test:
    stp x29, x30, [sp, #-16]!  // 保存 FP 和 LR 到栈
    mov x0, 1
    mov x1, 3
    bl csel_test
    ldp x29, x30, [sp], #16    // 从栈恢复 FP 和 LR
    ret
```

**优点**: 标准做法，支持多层嵌套
**缺点**: 需要栈操作

## 对比

| 方案 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| 临时寄存器 | 简单函数 | 快速 | 占用寄存器 |
| 栈保存 | 通用 | 标准、支持嵌套 | 需要栈操作 |
