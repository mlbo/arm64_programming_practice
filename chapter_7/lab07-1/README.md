# Lab 07-1: 栈与局部变量

## 实验目的

1. 理解栈帧的概念和作用
2. 掌握 STP/LDP 指令的使用
3. 学习局部变量的栈上存储

## 实验内容

1. **栈帧建立**: 使用 STP 保存 FP 和 LR
2. **局部变量**: 在栈上分配和访问局部变量
3. **栈帧恢复**: 使用 LDP 恢复 FP 和 LR

## 核心代码

```asm
// 建立栈帧
stp x29, x30, [sp, #-16]!   // 保存 FP 和 LR
mov x29, sp                  // 设置新的 FP

// 分配局部变量空间
sub sp, sp, #16              // 分配 16 字节

// 函数体...

// 恢复栈帧
mov sp, x29                  // 恢复 SP
ldp x29, x30, [sp], #16      // 恢复 FP 和 LR
ret
```
