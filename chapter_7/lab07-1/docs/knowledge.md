# 知识点：栈与栈帧

## 1. ARM64 栈特性

- 栈向下增长（高地址 → 低地址）
- SP 始终指向栈顶
- 栈必须 16 字节对齐

## 2. 关键寄存器

| 寄存器 | 名称 | 用途 |
|--------|------|------|
| sp | Stack Pointer | 栈指针 |
| x29 | Frame Pointer | 帧指针 |
| x30 | Link Register | 返回地址 |

## 3. STP/LDP 指令

### STP (Store Pair)

```asm
stp x29, x30, [sp, #-16]!   // 前变基：先减 16，再存储
stp x0, x1, [sp, #16]       // 偏移存储
```

### LDP (Load Pair)

```asm
ldp x29, x30, [sp], #16     // 后变基：先加载，再加 16
ldp x0, x1, [sp, #16]       // 偏移加载
```

## 4. 标准栈帧布局

```
高地址
+------------------+
| 返回地址 (x30)   |
+------------------+
| 保存的 FP (x29)  |  <- FP 指向这里
+------------------+
| 局部变量 1       |
+------------------+
| 局部变量 2       |
+------------------+  <- SP 指向这里
低地址
```

## 5. 栈帧建立步骤

1. 保存 FP 和 LR
2. 设置新的 FP
3. 分配局部变量空间
4. 函数体执行
5. 释放局部变量空间
6. 恢复 FP 和 LR
7. 返回
