## Chapter 10: 汇编进阶与内联汇编

### Labs: lab10-1 ~ lab10-4

**教学目标:** 内联汇编语法、原子操作入门

**核心内容:**
- lab10-1: 基础内联汇编
- lab10-2: 扩展内联汇编
- lab10-3: LDXR/STXR 原子操作
- lab10-4: LSE STSET (ARMv8.1-A)

**知识点:**
- `asm volatile (...)` 语法
- 输入/输出/破坏列表
- 内存屏障: dmb, dsb, isb
- LL/SC 原子模式

**C++ 设计:**
```cpp
// include/atomic.hpp
namespace benos {

template<typename T>
class Atomic {
public:
    explicit Atomic(T val = 0) : value_(val) {}

    auto load() const -> T {
        T result;
        asm volatile("ldxr %w0, [%1]"
                     : "=r"(result)
                     : "r"(&value_)
                     : "memory");
        return result;
    }

    void store(T val) {
        u32 tmp;
        asm volatile("1: stxr %w0, %w2, [%1]\n"
                     "   cbnz %w0, 1b"
                     : "=&r"(tmp)
                     : "r"(&value_), "r"(val)
                     : "memory");
    }

    auto fetch_add(T delta) -> T {
        T old_val, new_val;
        u32 tmp;
        asm volatile("1: ldxr %w0, [%3]\n"
                     "   add %w1, %w0, %w4\n"
                     "   stxr %w2, %w1, [%3]\n"
                     "   cbnz %w2, 1b"
                     : "=&r"(old_val), "=&r"(new_val), "=&r"(tmp)
                     : "r"(&value_), "r"(delta)
                     : "memory");
        return old_val;
    }

private:
    T value_;
};

} // namespace benos
```

## Chapter 11: 异常级别与异常向量表

### Labs: lab11-1 ~ lab11-3

**教学目标:** 理解 EL0-EL3、异常向量表、EL 切换

**核心内容:**
- lab11-1: 读取 CurrentEL
- lab11-2: 异常向量表建立
- lab11-3: EL1 → EL0 切换

**知识点:**
- CurrentEL, SPSel, ELR_EL1, SPSR_EL1
- 异常向量表布局 (11 对齐)
- Synchronous Exception vs IRQ
- eret 返回指令

**C++ 设计:**
```cpp
// include/arch/arm64/exception.hpp
namespace benos::arch::arm64 {

enum class ExceptionLevel : u32 {
    EL0 = 0,
    EL1 = 1,
    EL2 = 2,
    EL3 = 3,
};

auto current_el() -> ExceptionLevel {
    u64 el;
    asm volatile("mrs %0, CurrentEL" : "=r"(el));
    return static_cast<ExceptionLevel>((el >> 2) & 0x3);
}

// 异常向量表 (必须 2KB 对齐)
struct alignas(2048) ExceptionVector {
    u8 synchronous[128];      // Current EL, SP_EL0
    u8 irq[128];              // Current EL, SP_EL0
    u8 fiq[128];              // Current EL, SP_EL0
    u8 serror[128];           // Current EL, SP_EL0
    u8 synchronous_sp_elx[128];  // Current EL, SP_ELx
    u8 irq_sp_elx[128];
    u8 fiq_sp_elx[128];
    u8 serror_sp_elx[128];
    u8 lower_el_aarch64[128];   // Lower EL, AArch64
    u8 lower_el_irq_aarch64[128];
    u8 lower_el_fiq_aarch64[128];
    u8 lower_el_serror_aarch64[128];
    u8 lower_el_aarch32[128];   // Lower EL, AArch32
    u8 lower_el_irq_aarch32[128];
    u8 lower_el_fiq_aarch32[128];
    u8 lower_el_serror_aarch32[128];
};

void init_exception_vector();

} // namespace benos::arch::arm64
```
