## ncnn 代码规范要点

本章遵循 ncnn 风格的 C++ 设计：

### 内联汇编规范 (参考 ncnn 底层实现)
```cpp
// 内联汇编使用 asm volatile，明确标注 clobber 列表
template<typename T>
inline auto atomic_load(const T* addr) -> T {
    T result;
    asm volatile("ldxr %w0, [%1]"
                 : "=r"(result)
                 : "r"(addr)
                 : "memory");
    return result;
}
```

### 原子操作类设计 (参考 ncnn 简洁风格)
```cpp
template<typename T>
class Atomic {
public:
    explicit Atomic(T val = 0) : value_(val) {}

    auto load() const -> T;  // 返回值风格
    auto store(T val) -> void;  // void 返回
    auto fetch_add(T delta) -> T;  // 返回旧值

private:
    mutable T value_;  // mutable 支持 const load
};
```

- BL 指令修改 x30 (LR)

**C++ 设计:**
```cpp
// 内联汇编模板
template<typename T>
inline auto conditional_select(bool cond, T a, T b) -> T {
    T result;
    asm volatile("cmp %w3, #0; csel %w0, %w1, %w2, ne"
                 : "=r"(result)
                 : "r"(a), "r"(b), "r"(cond));
    return result;
}
```

## Chapter 6: 基础指令综合

### Labs: lab06-1 ~ lab06-4

**教学目标:** LDR/STR 深入, 原子操作入门

**核心内容:**
- lab06-1: LDR 多种寻址
- lab06-2: 数据处理深化
- lab06-3: 原子操作测试 (有 bug)
- lab06-4: 原子操作 (内核态)

**C++ 设计:**
```cpp
// include/atomic.hpp
namespace arm64lab {

template<typename T>
class Atomic {
public:
    explicit Atomic(T val = 0) : value_(val) {}

    auto load() const -> T {
        T result;
        asm volatile("ldxr %w0, [%1]"
                     : "=r"(result)
                     : "r"(&value_)
                     : "memory");
        return result;
    }

    void store(T val) {
        u32 tmp;
        asm volatile("1: stxr %w0, %w2, [%1]\n"
                     "   cbnz %w0, 1b"
                     : "=&r"(tmp)
                     : "r"(&value_), "r"(val)
                     : "memory");
    }

    auto fetch_add(T delta) -> T;

private:
    T value_;
};

} // namespace arm64lab
```

## Chapter 8: C 与汇编混合编程

### Labs: lab08-1 ~ lab08-4

**教学目标:** C 调汇编、汇编调 C、ARM64 ABI

**核心内容:**
- lab08-1: 基础比较测试
- lab08-2: C 调汇编
- lab08-3: 汇编调 C
- lab08-4: 完整裸机项目

**ARM64 ABI 要点:**
- 参数: x0-x7
- 返回值: x0
- 被调用者保存: x19-x28, x29, x30
- 栈 16 字节对齐

**C++ 设计:**
```cpp
// extern "C" 声明汇编函数
extern "C" {
    int compare_data(int a, int b);  // 汇编实现
}

// 或使用内联汇编
inline auto compare_data_cpp(int a, int b) -> int {
    int result;
    asm volatile("cmp %1, %2; csel %0, %1, %2, hi"
                 : "=r"(result)
                 : "r"(a), "r"(b));
    return result;
}
```
