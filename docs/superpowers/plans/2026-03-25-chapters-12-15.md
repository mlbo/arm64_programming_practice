## ncnn 代码规范要点

本章遵循 ncnn 风格的 C++ 设计：

### 驱动类设计 (参考 ncnn Layer 基类风格)
```cpp
namespace benos {

// 基础驱动接口
class IrqController {
public:
    virtual ~IrqController() = default;

    virtual auto init() -> int = 0;  // 0=成功, 负数=错误
    virtual auto enable_irq(u32 irq_id) -> int = 0;
    virtual auto disable_irq(u32 irq_id) -> int = 0;
};

// 具体实现
class GicV2 : public IrqController {
public:
    auto init() -> int override;
    auto enable_irq(u32 irq_id) -> int override;
    auto disable_irq(u32 irq_id) -> int override;
    auto acknowledge() -> u32;  // 返回 IRQ ID

private:
    static constexpr u64 GICD_BASE = 0xFF841000;
    static constexpr u64 GICC_BASE = 0xFF842000;
};

} // namespace benos
```

### POD 结构体 (与汇编接口匹配)
```cpp
// 必须是 POD 类型，与汇编 kernel_entry/kernel_exit 匹配
struct PtRegs {
    u64 x0, x1, x2, x3, x4, x5, x6, x7;
    u64 x8, x9, x10, x11, x12, x13, x14, x15;
    u64 x16, x17, x18, x19, x20, x21, x22, x23;
    u64 x24, x25, x26, x27, x28, x29, x30;
    u64 sp, pc, pstate;
};

static_assert(sizeof(PtRegs) == 31 * 8 + 24, "PtRegs size mismatch");
static_assert(std::is_standard_layout_v<PtRegs>, "PtRegs must be POD");
```

## Chapter 13: GIC 中断控制器

### Lab: lab13-1

**教学目标:** GICv2 驱动实现

**核心内容:**
- GIC Distributor 初始化
- GIC CPU Interface 初始化
- 中断使能/禁用
- EOI (End of Interrupt)

**知识点:**
- GICv2 架构: Distributor + CPU Interface
- 中断 ID: SGI (0-15), PPI (16-31), SPI (32+)
- 优先级、目标 CPU

**C++ 设计:**
```cpp
// include/driver/gicv2.hpp
namespace benos {

class GicV2 {
public:
    void init();
    void enable_irq(u32 irq_id);
    void disable_irq(u32 irq_id);
    void eoi(u32 irq_id);
    auto acknowledge() -> u32;  // 返回 IRQ ID

    static auto instance() -> GicV2&;

private:
    GicV2() = default;

    // GICv2 寄存器基址
    static constexpr u64 GICD_BASE = 0xFF841000;  // Distributor
    static constexpr u64 GICC_BASE = 0xFF842000;  // CPU Interface

    void init_distributor();
    void init_cpu_interface();
};

extern GicV2& g_gic;

} // namespace benos
```

## Chapter 14: 中断与定时器管理

### Labs: lab14-1 ~ lab14-2

**教学目标:** GIC + Timer 集成、中断处理流程

**核心内容:**
- lab14-1: 定时器中断
- lab14-2: 中断 + 页表 (综合)

**知识点:**
- ARM Generic Timer
- CNTP_CTL_EL0, CNTP_TVAL_EL0
- Timer 中断路由

**C++ 设计:**
```cpp
// include/driver/timer.hpp
namespace benos {

class TimerDriver {
public:
    void init();
    void handle_irq();
    void set_interval(u64 ns);

    static constexpr u64 DEFAULT_INTERVAL = 1000000000ULL / 100;  // 10ms

private:
    TimerDriver() = default;
    u64 interval_ns_{DEFAULT_INTERVAL};
    u64 tick_count_{0};
};

extern TimerDriver& g_timer;

} // namespace benos

// include/irq_manager.hpp
namespace benos {

using IrqHandler = void (*)(void*);

class IrqManager {
public:
    void init();
    void register_handler(u32 irq_id, IrqHandler handler, void* data);
    void handle_irq();

    static auto instance() -> IrqManager&;

private:
    IrqManager() = default;

    static constexpr u32 MAX_IRQ = 256;
    struct IrqEntry {
        IrqHandler handler{nullptr};
        void* data{nullptr};
    };
    IrqEntry entries_[MAX_IRQ];
};

extern IrqManager& g_irq_manager;

} // namespace benos
```

## Chapter 15: 缓存管理与页表遍历

### Labs: lab15-1 ~ lab15-2

**教学目标:** 缓存层次结构、缓存维护操作

**核心内容:**
- lab15-1: 缓存信息查询
- lab15-2: 页表遍历 (地址转换)

**知识点:**
- CTR_EL0: 缓存类型寄存器
- CLIDR_EL1: 缓存层次 ID 寄存器
- dc civac, dc ivac, dc cvac
- CCSIDR_EL1: 缓存大小 ID 寄存器

**C++ 设计:**
```cpp
// include/arch/arm64/cache.hpp
namespace benos::arch::arm64 {

struct CacheInfo {
    u32 line_size;      // 缓存行大小 (字节)
    u32 sets;           // 组数
    u32 ways;           // 路数
    u32 level;          // 缓存级别
};

class CacheMaintainer {
public:
    static auto get_cache_info(u32 level, bool is_instruction) -> CacheInfo;
    static void clean_by_va(u64 va);
    static void invalidate_by_va(u64 va);
    static void clean_and_invalidate_by_va(u64 va);
    static void clean_all();
    static void invalidate_all();

private:
    static auto read_ccsidr(u32 level, bool is_instruction) -> u64;
};

// 页表遍历
class PageTableWalker {
public:
    static auto walk(u64 va) -> u64;  // 返回物理地址
    static auto get_pa(u64 va) -> u64;
};

} // namespace benos::arch::arm64
```
