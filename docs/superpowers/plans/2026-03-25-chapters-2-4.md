# lab03-3: 前/后变基寻址模式

## 前变基 (Pre-indexed)
`ldr x6, [x1, #8]!`
- 先更新地址: x1 = x1 + 8
- 再加载值

## 后变基 (Post-indexed)
`ldr x7, [x1], #8`
- 先加载值
- 再更新地址: x1 = x1 + 8

## 应用场景
- 遍历数组
- 内存拷贝
```

// include/bit_ops.hpp
namespace arm64lab::bit {

template<typename T>
constexpr auto extract(T val, u32 lsb, u32 width) -> T {
    return (val >> lsb) & ((T(1) << width) - 1);
}

template<typename T>
constexpr auto insert(T dst, T src, u32 lsb, u32 width) -> T {
    T mask = ((T(1) << width) - 1) << lsb;
    return (dst & ~mask) | ((src << lsb) & mask);
}

} // namespace arm64lab::bit
```
