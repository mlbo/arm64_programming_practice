## Chapter 22: NEON SIMD

### Labs: lab22-1 ~ lab22-3

**教学目标:** NEON intrinsics、自动向量化

**核心内容:**
- lab22-1: RGB/BGR 转换
- lab22-2: 矩阵乘法
- lab22-3: 自动向量化对比

**知识点:**
- NEON 寄存器: V0-V31 (128-bit)
- 向量类型: int8x8_t, float32x4_t, uint8x16_t
- 交错加载/存储: vld3q_u8, vst3q_u8
- FMA: vfmaq_laneq_f32

**C++ 设计:**
```cpp
// include/neon_utils.hpp
namespace arm64lab::neon {

// RGB → BGR 转换 (使用 NEON intrinsics)
void rgb_to_bgr_neon(const u8* src, u8* dst, size_t pixels);

// 矩阵乘法 4x4 (NEON 优化)
void matrix_mul_4x4_neon(const float* a, const float* b, float* result);

// 标量版本对比
void rgb_to_bgr_scalar(const u8* src, u8* dst, size_t pixels);
void matrix_mul_4x4_scalar(const float* a, const float* b, float* result);

// 自动向量化测试
void rgb_to_bgr_auto_vectorized(const u8* src, u8* dst, size_t pixels);

} // namespace arm64lab::neon

// 实现
namespace arm64lab::neon {

void rgb_to_bgr_neon(const u8* src, u8* dst, size_t pixels) {
    size_t i = 0;

    // 每次处理 16 像素 (48 字节)
    for (; i + 15 < pixels; i += 16) {
        // 交错加载 RGB
        uint8x16x3_t rgb = vld3q_u8(src + i * 3);

        // 交换 R 和 B
        uint8x16x3_t bgr;
        bgr.val[0] = rgb.val[2];  // B
        bgr.val[1] = rgb.val[1];  // G
        bgr.val[2] = rgb.val[0];  // R

        // 交错存储 BGR
        vst3q_u8(dst + i * 3, bgr);
    }

    // 处理剩余像素
    for (; i < pixels; ++i) {
        dst[i * 3 + 0] = src[i * 3 + 2];  // B
        dst[i * 3 + 1] = src[i * 3 + 1];  // G
        dst[i * 3 + 2] = src[i * 3 + 0];  // R
    }
}

void matrix_mul_4x4_neon(const float* a, const float* b, float* result) {
    // 加载矩阵 B 的列
    float32x4_t b0 = vld1q_f32(b);
    float32x4_t b1 = vld1q_f32(b + 4);
    float32x4_t b2 = vld1q_f32(b + 8);
    float32x4_t b3 = vld1q_f32(b + 12);

    // 计算结果矩阵的每一行
    for (int i = 0; i < 4; ++i) {
        float32x4_t a_row = vld1q_f32(a + i * 4);
        float32x4_t r = vmulq_laneq_f32(b0, a_row, 0);
        r = vfmaq_laneq_f32(r, b1, a_row, 1);
        r = vfmaq_laneq_f32(r, b2, a_row, 2);
        r = vfmaq_laneq_f32(r, b3, a_row, 3);
        vst1q_f32(result + i * 4, r);
    }
}

} // namespace arm64lab::neon
```

## Chapter 23: SVE SIMD

### Labs: lab23-1 ~ lab23-6

**教学目标:** SVE predicate、VLA 编程

**核心内容:**
- lab23-1: SVE 基础与 predicate
- lab23-2: 交错加载/存储 (ld3b/st3b)
- lab23-3: memcpy 实现
- lab23-4: strcmp 实现
- lab23-5: 矩阵乘法
- lab23-6: FFR (First Fault Register)

**知识点:**
- SVE 向量长度可变 (128-2048 bit)
- Predicate 寄存器: P0-P15
- whilelo, ptrue 指令
- VLA (Vector Length Agnostic) 编程

**C++ 设计:**
```cpp
// include/sve_utils.hpp
namespace arm64lab::sve {

// 检查 SVE 支持
auto has_sve() -> bool {
    u64 id_aa64pfr0;
    asm volatile("mrs %0, ID_AA64PFR0_EL1" : "=r"(id_aa64pfr0));
    return (id_aa64pfr0 >> 32) & 0xF;  // SVE field
}

// 获取当前向量长度 (字节)
auto get_sve_vector_length() -> u64 {
    u64 vl;
    asm volatile("rdvl %0, #1" : "=r"(vl));
    return vl;
}

// SVE memcpy
void memcpy_sve(void* dst, const void* src, size_t n);

// SVE strcmp
auto strcmp_sve(const char* s1, const char* s2) -> int;

// SVE RGB/BGR 转换
void rgb_to_bgr_sve(const u8* src, u8* dst, size_t pixels);

} // namespace arm64lab::sve

// 汇编实现 (SVE 指令在 C++ intrinsics 中支持有限)
// src/memcpy_sve.S
/*
memcpy_sve:
    mov x2, x0          // 保存目标
    mov x0, #0
    ptrue p0.b          // 全 1 predicate
1:
    whilelo p1.b, x0, x1
    b.none 2f
    ld1b z0.b, p1/z, [x0]
    st1b z0.b, p1, [x2, x0]
    incb x0
    b 1b
2:
    ret
*/
```
