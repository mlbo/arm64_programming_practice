# Lab 03-2: 前/后变基寻址

## 实验目标

学习 LDR/STR 指令的前变基和后变基寻址模式。

## 编译与运行

```bash
make
make run
```

## 核心内容

### 前变基 (Pre-indexed)

```asm
ldr x6, [x1, #8]!
```

1. 先更新地址: `x1 = x1 + 8`
2. 再加载数据

### 后变基 (Post-indexed)

```asm
ldr x7, [x1], #8
```

1. 先加载数据
2. 再更新地址: `x1 = x1 + 8`

## 应用场景

- 遍历数组
- 内存拷贝
- 链表遍历
