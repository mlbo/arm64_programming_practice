# 知识点: LDR 偏移模式

## LDR 指令格式

```
LDR Xt, [Xn{, #imm}]
LDR Xt, [Xn, Xm{, LSL #shift}]
```

## 偏移模式

### 1. 无偏移

```asm
ldr x0, [x1]
```

直接从 `x1` 指向的地址加载 64 位数据到 `x0`。

### 2. 立即数偏移

```asm
ldr x0, [x1, #8]
```

从 `x1 + 8` 地址加载数据。偏移量范围: -256 ~ +255 (字节)。

### 3. 寄存器偏移

```asm
ldr x0, [x1, x2]
```

从 `x1 + x2` 地址加载数据。

### 4. 移位偏移

```asm
ldr x0, [x1, x2, lsl #3]
```

从 `x1 + (x2 << 3)` 地址加载数据。

| 移位量 | 说明 |
|--------|------|
| LSL #0 | 不移位 (默认) |
| LSL #1 | 乘以 2 |
| LSL #2 | 乘以 4 |
| LSL #3 | 乘以 8 |

## 应用场景

### 数组访问

```asm
// 访问 long 数组的第 n 个元素
ldr x0, [x1, x2, lsl #3]  // array[n]
```

### 结构体访问

```asm
// 访问结构体成员
ldr x0, [x1, #16]  // 偏移 16 字节的成员
```

## 注意事项

- 偏移量必须在允许范围内
- 移位量只能是 0, 1, 2, 3
- 这些模式不会修改基址寄存器
