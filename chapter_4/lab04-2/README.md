# Lab 04-2: ADDS/CMP/ADC 指令

## 实验目标

学习 ADDS、CMP、ADC 指令及其对条件标志位的影响。

## 编译与运行

```bash
make
make run
```

## 核心内容

### ADDS 指令

```asm
adds x0, x1, x1   // 加法并设置标志位
```

- 结果为 0 时 Z=1
- 发生溢出时 C=1

### ADC 指令

```asm
adc x3, xzr, xzr  // 带进位加法
```

可以用来捕获进位值。

### CMP 指令

```asm
cmp x1, x2        // 比较
```

| 比较 | N | Z | C | V |
|------|---|---|---|---|
| x1 > x2 | ? | 0 | 1 | ? |
| x1 < x2 | ? | 0 | 0 | ? |
| x1 == x2 | 0 | 1 | 1 | 0 |
