# 练习题

## 基础题

### 1. 建立异常向量表

创建一个基本的异常向量表，并在每个入口点打印错误信息。

### 2. 触发异常

编写代码触发一个同步异常，观察异常处理流程：

```cpp
void trigger_alignment() {
    // 访问未对齐地址
}
```

## 进阶题

### 3. 解析 ESR

实现解析 ESR_EL1 寄存器的函数：

```cpp
void decode_esr(u32 esr) {
    // 解析 EC 和 ISS
}
```

### 4. 打印寄存器状态

在异常处理函数中打印所有通用寄存器的值。

## 挑战题

### 5. 实现断点

使用 BRK 指令实现软件断点支持。
