# 练习题

## 基础题

### 1. 实现 switch_to_el0

编写从 EL1 切换到 EL0 的函数。

### 2. 设置用户栈

实现设置 EL0 栈指针的代码。

## 进阶题

### 3. 从 EL3 启动

编写完整的启动代码，从 EL3 逐步下降到 EL1，再到 EL0。

### 4. 系统调用框架

实现基本的系统调用处理：
- 在 EL0 使用 `svc #0`
- 在 EL1 的异常向量表中处理 SVC

## 挑战题

### 5. 简单的用户程序

创建一个简单的用户程序，在 EL0 中打印消息（通过系统调用）。

### 6. 进程切换

实现两个用户程序之间的切换（简单的进程调度）。
