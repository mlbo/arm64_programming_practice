# 内联汇编知识点

## 1. 基本语法

### asm 关键字

GCC 提供了 `asm` 关键字用于在 C/C++ 代码中嵌入汇编：

```cpp
asm ("汇编代码");
```

### volatile 关键字

`volatile` 告诉编译器不要优化这段汇编代码：

```cpp
asm volatile ("nop");  // 不会被优化掉
```

### 完整语法

```cpp
asm volatile (
    "汇编模板"
    : 输出操作数列表      // 可选
    : 输入操作数列表      // 可选
    : 破坏列表            // 可选
);
```

## 2. 操作数约束

### 常用约束符

| 约束符 | 含义 |
|--------|------|
| `r` | 通用寄存器 |
| `w` | 32位寄存器 (w0-w30) |
| `x` | 64位寄存器 (x0-x30) |
| `m` | 内存地址 |
| `i` | 立即数 |
| `=` | 只写 |
| `+` | 读写 |

### 示例

```cpp
// 输出约束
int result;
asm volatile (
    "mov %w0, #42"
    : "=r" (result)   // = 表示只写
);

// 输入约束
int a = 10;
asm volatile (
    "add %w0, %w0, #1"
    : "+r" (a)        // + 表示读写
);

// 内存操作数
int value = 0;
asm volatile (
    "str %w0, [%1]"
    :                 // 无输出
    : "r" (42), "m" (value)
);
```

## 3. 操作数修饰符

ARM64 特有的修饰符：

| 修饰符 | 含义 | 示例 |
|--------|------|------|
| `%wN` | 32位寄存器 | `%w0` -> w0 |
| `%xN` | 64位寄存器 | `%x0` -> x0 |
| `%cN` | 常量值 | `%c0` -> 立即数 |

```cpp
int value = 42;
asm volatile (
    "mov %w0, %w1"    // 使用 32 位寄存器
    : "=r" (result)
    : "r" (value)
);
```

## 4. 破坏列表

告诉编译器哪些寄存器或内存被修改了：

```cpp
asm volatile (
    "ldr x0, [%0]\n"
    "add x0, x0, #1\n"
    "str x0, [%0]"
    :
    : "r" (ptr)
    : "x0", "memory"  // x0 和内存被修改
);
```

### 常用破坏项

- `"memory"`: 内存被修改
- `"cc"`: 条件标志被修改
- 寄存器名: `"x0"`, `"x1"`, ...
