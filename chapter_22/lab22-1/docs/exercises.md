# 练习题

## 基础练习

### 1. 修改像素处理数量

当前代码每次处理 16 像素。请修改代码，使其每次处理 8 像素，并比较性能差异。

**提示**: 使用 `uint8x8x3_t` 和 `vld3_u8` / `vst3_u8`。

### 2. 添加 RGBA 到 BGRA 转换

扩展代码，实现 RGBA (4 通道) 到 BGRA 的转换。

**提示**: 使用 `vld4q_u8` 和 `vst4q_u8` 处理 4 通道数据。

## 进阶练习

### 3. 实现图像灰度化

使用 NEON 实现 RGB24 到灰度图的转换。

灰度计算公式：
```
gray = 0.299 * R + 0.587 * G + 0.114 * B
```

**提示**: 使用定点数运算避免浮点：
```
gray = (77 * R + 150 * G + 29 * B) >> 8
```

### 4. 实现图像亮度调整

使用 NEON 实现图像亮度调整功能。

```cpp
void adjust_brightness_neon(
    const u8* src, u8* dst, usize pixels, int delta
);
```

**提示**: 使用 `vqaddq_u8` (饱和加法) 防止溢出。

### 5. 性能优化挑战

尝试优化现有代码，使 NEON 版本比标量版本快 10 倍以上。

**优化思路**:
- 循环展开
- 预取指令
- 多线程处理
