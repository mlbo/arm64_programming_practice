# NEON 矩阵乘法知识

## 矩阵乘法基础

### 4x4 矩阵乘法公式

对于 C = A × B，其中 A, B, C 都是 4x4 矩阵：

```
C[i][j] = Σ(k=0 to 3) A[i][k] × B[k][j]
```

标量实现需要：
- 4 × 4 × 4 = 64 次乘法
- 4 × 4 × 3 = 48 次加法
- 总计 112 次浮点运算

## NEON 优化策略

### 列主序存储

NEON 加载的是列向量，因此使用列主序存储更高效：

```
内存布局 (列主序):
A[0][0], A[1][0], A[2][0], A[3][0],  // 第0列
A[0][1], A[1][1], A[2][1], A[3][1],  // 第1列
...
```

### 利用 FMA 指令

FMA (Fused Multiply-Add) 在一条指令中完成 `a × b + c`：

```cpp
// 传统方式 (2 条指令)
r = vmulq_f32(a, b);    // 乘法
r = vaddq_f32(r, c);    // 加法

// FMA 方式 (1 条指令)
r = vfmaq_f32(c, a, b); // 融合乘加
```

### vfmaq_laneq_f32 指令

`vfmaq_laneq_f32(r, a, b, lane)` 的功能：
- 从向量 `b` 中提取第 `lane` 个元素
- 与向量 `a` 的所有元素相乘
- 结果与 `r` 相加

```cpp
float32x4_t a = {a0, a1, a2, a3};
float32x4_t b = {b0, b1, b2, b3};
float32x4_t r = {r0, r1, r2, r3};

// 使用 b 的第 0 个元素
r = vfmaq_laneq_f32(r, a, b, 0);
// 结果: r = {r0 + a0*b0, r1 + a1*b0, r2 + a2*b0, r3 + a3*b0}
```

## 优化后的矩阵乘法

### 算法思路

1. 加载矩阵 B 的 4 列到 4 个向量寄存器
2. 对于矩阵 A 的每一行：
   - 加载该行到 1 个向量寄存器
   - 使用 4 次 FMA 计算结果行
3. 存储结果行

### 指令数量对比

| 版本 | 加载 | 计算 | 存储 |
|------|------|------|------|
| 标量 | 32 | 112 | 16 |
| NEON | 8 + 4 = 12 | 16 FMA | 4 |
| ASM | 8 | 16 FMLA | 4 |

## 性能考虑

### 流水线利用

NEON 指令通常有多个执行单元，可以并行执行：
- 尽量减少数据依赖
- 使用循环展开提高指令级并行度

### 内存访问

- 尽量使用连续内存访问
- 考虑缓存行对齐 (64 字节)
